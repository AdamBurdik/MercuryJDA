# MercuryJDA
MercuryJDA is a library that provides useful features for working with [JDA](https://github.com/discord-jda/JDA).

Features:
- Easy creation of message templates with buttons and select menus
- Easy creation of modal dialogs
- Message configuration with user-specific context and arguments

# How To Build
1. Clone the repository
```bash
git clone https://github.com/adamBurdik/MercuryJDA
```
2. Navigate to the directory
```bash
cd MercuryJDA
```
3. Build the module
```bash
gradlew build
```
